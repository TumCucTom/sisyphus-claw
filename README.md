# Sisyphus workspace

OpenClaw agent personality: **Sisyphus**. Drop this folder into your OpenClaw home (e.g. `~/.openclaw/workspace-sisyphus/`) or point your agent at it. Tuned for fitness, training, and daily habits.

**About the bot:** Fitness and habits coach — tracks training, nutrition, sleep, and goals; keeps you consistent without being pushy. Runs on [OpenClaw](https://openclaw.ai).

## Add a USER file

`USER.md` is not in the repo (it’s private). Create it locally so the agent knows who it’s helping.

**Example format:**

```markdown
# USER.md - About Your Human

- **Name:**
- **What to call them:**
- **Pronouns:** _(optional)_
- **Timezone:**

## Background

- **Athletic history:** _(e.g. sports, level)_
- **Current goals:** _(e.g. race times, strength targets)_
- **Current focus:** _(e.g. weight loss, muscle, endurance)_
- **Injuries / constraints:** _(what to work around)_

## Body stats _(optional)_

- **Weight:**
- **Height:**

## Other interests

_(Climbing, mobility, etc. — whatever helps the agent support you.)_

---

Update as we go.
```
